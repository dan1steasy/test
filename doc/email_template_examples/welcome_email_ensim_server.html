1. Appliance administrator details (top level)

Within the appliance administration (the Ensim Control Panel), you will have access to server-wide configuration options. You will be able to add new sites to the server, view bandwidth usage for individual sites and configure services such as the web and mail servers. Comprehensive help files are built into the interface to give you more information.

  • Your System Admin User Name: [[SERVER_ADMIN_USER]]
  • Your System Admin Password: [[SERVER_ADMIN_PASS]]
  • Appliance admin site url: http://[[FQDN]]/admin

You can safely ignore any warning messages that your browser may display about the SSL certificate on the admin interface: your browser session is protected, but the certificate is self-signed by the server, not an external authority.

2. Site administrator details (hosting account level)

A site administrator interface is available to the admin user for each site. Within the site administration pages you will be able to carry out many tasks, including: adding users to sites, editing user email settings (e.g. auto-responders, forwarding, adding/deleting aliases), viewing web statistics and password-protecting directories.

  • Site admin site: http://www.domainname/admin (where domainname will be one of the sites set up on the server)
  • Login name: The site admin for that particular site
  • You can also get to the site admin pages for a particular site by clicking on its domain name via the 'View Sites' page of the main Appliance Admin Site.

3. User administrator details (individual user level)

For each additional user that is added to a site by the site administrator, there is a user interface available. This area enables each user to set up email forwarding, aliases, auto-responders and spam settings.

  • User admin site: http://www.domainname.com/user
